## 📦 Modules

### Module 1 — MedJarvis
Assistant médical IA pour étudiants en médecine.
- Génération de fiches de révision
- Générateur de QCM
- Système de révisions espacées (Anki-style)
- Base de connaissances médicale

### Module 2 — SmartCold
Gestion intelligente de conservateur médical (chaîne du froid).
- Suivi de température en temps réel
- Système d'alertes
- Historique et statistiques

### Module 3 — Chariow Academy
Ressources islamiques et formations digitales.
- Rappels islamiques quotidiens
- Bibliothèque de Dou'as
- Articles & quiz
- Bibliothèque PDF

### Module 4 — AI Workspace
Suite d'agents IA pour la productivité.
- Agent de recherche
- Agent de rédaction
- Agent d'analyse
- Gestion documentaire
